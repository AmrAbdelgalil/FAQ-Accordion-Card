# FAQ-Accordion-Card

Building a responsive FAQ Accordion Card using HTML, CSS, Media query and JavaScript.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- mediaquery
- JavaScript to build Accordion function from scratch.
